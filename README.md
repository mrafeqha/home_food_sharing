# Home Food Sharing Platform

## Overview

Home Food Sharing Platform is a web application that connects home cooks with customers looking for fresh, homemade meals. Users can browse available meals, post food listings, and manage their accounts through a simple and user-friendly interface.

## Features

* User Registration and Login
* Browse Available Meals
* Post Homemade Food Listings
* View Meal Details
* Responsive User Interface

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* XAMPP

## Setup Instructions

1. Install XAMPP.
2. Copy the project folder into `htdocs`.
3. Start Apache and MySQL.
4. Import `db.sql` into phpMyAdmin.
5. Open:
   http://localhost/home_food_sharing/frontend/
